# use-debounce-search

A simple React project demonstrating a **search input with debouncing** using a **custom hook** to improve performance and reduce unnecessary operations while typing.

## Features
- Custom `useDebounce` hook
- Optimized search input handling
- Functional React components with hooks
- Clean and reusable logic

## Tech Stack
- React
- JavaScript
- Hooks (useState, useEffect)

## Use Case
Debouncing is useful in scenarios like search boxes to avoid triggering logic or API calls on every keystroke.
